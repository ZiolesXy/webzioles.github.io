Pembahasan Dasar JavaScript

JavaScript adalah bahasa pemrograman yang sering digunakan untuk membuat halaman web menjadi interaktif. Beberapa fitur dasar JavaScript yang dicoba di project ini meliputi:

  • Manipulasi DOM (Document Object Model)
  • Menangani peristiwa (event)
  • Menangani variabel, tipe data, dan operasi logika
  • Penggunaan fungsi
  • Penggunaan kondisional seperti if-else statement
  • Penggunaan loop seperti for dan while
